# flaffylter

Use snapchat like  filters while maintaning your privacy.

- Use facial filters like:
  - glasses
  - hats
  - facial makeup
  - faceswap
  - face aging
 - You face images will be encrypted.
 - While in the server image  cannot  reveal your face.
